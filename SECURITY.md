# Security Policy

## Supported Versions

Only the current version of the MUCwahl is supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability, please do NOT create a public issue. Instead, please report it directly to the maintainer:

**Frederik Metz**
Email: frederikmetz@gmail.com

We will acknowledge your report within 48 hours and work on a fix as soon as possible.

## Our Security Philosophy
This project follows a "Fortress Zero" architecture:
- Static-only (HTML/JS)
- No backend / No database
- No cookies / No tracking
- Client-side processing only
